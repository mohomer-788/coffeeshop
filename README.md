CoffeeShop Project

This is a simple web project for a coffee shop. It shows products from a database and allows users to add items to a cart.

Live site:
https://coffeeshop-6ar2.onrender.com/products.php

Technologies used:
PHP
MySQL
HTML
CSS

Features:
- Show products from database
- Products grouped by category
- Add to cart
- Session cart system
- Connect to MySQL database

Database:
products table:
id, name, price, category, image

orders table:
id, total, order_date

How to run:
- Import database file into MySQL
- Update db.php with database info
- Run using local server (XAMPP or Laragon)
